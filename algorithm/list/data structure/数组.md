# 数组

## 双指针

常见思路：左右指针 和 快慢指针
* 快慢指针

[remove-element](https://leetcode.cn/problems/remove-element/)

```java
public int removeDuplicates(int[] nums) {
    if (nums.length <= 1) return nums.length;
    // 慢指针
    int slow = 0;
    // 快指针
    int fast = 0;
    while (fast < nums.length) {
        if (nums[slow] != nums[fast]) {
            slow++;
            nums[slow] = nums[fast];
        }
        fast++;
    }
    return slow + 1;
}
```

* 左右指针

[two-sum-ii-input-array-is-sorted](https://leetcode.cn/problems/two-sum-ii-input-array-is-sorted/)

```java
public int[] twoSum(int[] numbers, int target) {
    // 左指针
    int left = 0;
    // 右指针
    int right = numbers.length - 1;

    while(left < right){
        int tmp = numbers[left] + numbers[right];
        if(tmp > target) {
            right--;
        }else if(tmp < target){
            left++;
        }else {
            return new int[]{left + 1, right + 1};
        }
    }

    return new int[]{-1, -1};
}
```

## 前缀和

* 一维数组

(range-sum-query-immutable)[https://leetcode.cn/problems/range-sum-query-immutable/]

```java
class NumArray {
    // 前缀和数组
    private int[] prefixSum;

    public NumArray(int[] nums) {
        prefixSum = new int[nums.length + 1];
        for(int i = 1; i <= nums.length; i++){
            prefixSum[i] = prefixSum[i - 1] + nums[i - 1];
        }
    }
    
    public int sumRange(int left, int right) {
        return prefixSum[right + 1] - prefixSum[left];
    }
}
```

* 二维数组

(range-sum-query-2d-immutable)[https://leetcode.cn/problems/range-sum-query-2d-immutable/]

```java
class NumMatrix {
    // 前缀和数组
    private int[][] prefixSum;
    public NumMatrix(int[][] matrix) {
        prefixSum = new int[matrix.length + 1][matrix[0].length + 1];
        for(int i = 1; i <= matrix.length; i++){
            for(int j = 1; j <= matrix[0].length; j++){
                prefixSum[i][j] = prefixSum[i - 1][j] + prefixSum[i][j - 1] - prefixSum[i - 1][j - 1] + matrix[i - 1][j - 1];
            }
        }
    }
    
    public int sumRegion(int row1, int col1, int row2, int col2) {
        return prefixSum[row2 + 1][col2 + 1] + prefixSum[row1][col1] - prefixSum[row2 + 1][col1] - prefixSum[row1][col2 + 1];
    }
}
```

## 差分数组

场景：频繁对原始数组的某个区间元素进行增减
* diff[i] = nums[i] - nums[i - 1]

```java
// 差分数组工具类
class Difference {
    // 差分数组
    private int[] diff;
    
    // 输入一个初始数组，区间操作将在这个数组上进行
    public Difference(int[] nums) {
        diff = new int[nums.length];
        // 根据初始数组构造差分数组
        diff[0] = nums[0];
        for (int i = 1; i < nums.length; i++) {
            diff[i] = nums[i] - nums[i - 1];
        }
    }

    // 给闭区间 [i, j] 增加 val（可以是负数）
    public void increment(int i, int j, int val) {
        diff[i] += val;
        if (j + 1 < diff.length) {
            diff[j + 1] -= val;
        }
    }

    // 返回结果数组
    public int[] result() {
        int[] res = new int[diff.length];
        // 根据差分数组构造结果数组
        res[0] = diff[0];
        for (int i = 1; i < diff.length; i++) {
            res[i] = res[i - 1] + diff[i];
        }
        return res;
    }
}

```

## 二分搜索

```java
/**
* 二分搜索
*/
public int search(int[] nums, int target) {
    int l = 0;
    int r = nums.length - 1;

    while (l <= r) {
        int mid = (l + r) / 2;
        if (nums[mid] > target) {
            r = mid - 1;
        } else if (nums[mid] < target) {
            l = mid + 1;
        } else {
            return mid;
        }
    }
    return -1;
}

/**
* 二分搜索-左边界
*/
private int searchLeft(int[] nums, int target) {
    if (nums.length == 0) return -1;

    int l = 0;
    int r = nums.length - 1;

    while (l < r) {
        int mid = l + (r - l) / 2;

        if (nums[mid] > target) {
            r = mid - 1;
        } else if (nums[mid] < target) {
            l = mid + 1;
        } else if (nums[mid] == target) {
            // 右边收缩
            r = mid;
        }
    }
    return l;
}

/**
* 二分搜索-右边界
*/
private int searchRight(int[] nums, int target) {
    if (nums.length == 0) return -1;

    int l = 0;
    int r = nums.length - 1;

    while (l <= r) {
        int mid = l + (r - l) / 2;

        if (nums[mid] > target) {
            r = mid - 1;
        } else if (nums[mid] < target) {
            l = mid + 1;
        } else if (nums[mid] == target) {
            // 左边收缩
            l = mid;
        }
    }

    return r;
}
```
